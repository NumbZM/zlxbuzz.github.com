    # 居中

    # 水平居中

    1. 块级元素包裹行内元素(span,inline-block)，直接给外层 text-align: center
      水平居中
    <style>
.text-center{
  text-align: center;
}
</style>
<div class="p text-center">
    <span class="c">水平居中</span>
</div>
    1. 块级元素包含有一定宽度的块级元素
      水平居中
    <style>
.block{
  width: 300px;
  margin: 0 auto;
  text-align: center;
}
</style>
<div class="p">
    <div class="c block">水平居中</div>
</div>

    3.flex

    水平居中

    <style>
.flex-center{
  display: flex;
  justify-content: center;
}
</style>
<div class="p flex-center">
    <div class="c">水平居中</div>
</div>
    1. transform基于父级居中,但是父级会塌陷
      水平居中
    <style>
.transform{
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
</style>
<div class="p relative">
    <div class="c transform">水平居中</div>
</div>

    5.绝对定位元素也可以使用margin，兼容更加好

    水平居中

    <style>
.absolute{
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
  width: 4em;
}
</style>
<div class="p relative">
    <div class="c absolute">水平居中</div>
</div>

    # 垂直居中

    1.单行文本采用line-height

    垂直居中 

    <style>
.vertical{
  height: 50px;
  line-height: 50px;
}
</style>
<div class="p vertical">
    垂直居中
</div>

    2.flex

    垂直居中 

    <style>
.flex-vertical{
  height: 50px;
  display: flex;
  align-items: center;
}
</style>
<div class="p flex-vertical">
    垂直居中
</div>

    3.transform基于父级居中

    垂直居中

    <style>
.transform-v{
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
</style>
<div class="p relative">
    <div class="c transform-v">垂直居中</div>
</div>

    4.table-cell和vertical-align,可以多行文字居中

    垂直居中
    垂直居中
    <style>
.table-cell{
  display: table-cell;
  height: 100px;
}
.vertical-align{
  vertical-align: middle;
}
</style>
<div class="p table-cell vertical-align">
    <div class="c">垂直居中</div>
    <div class="c">垂直居中</div>
</div>
