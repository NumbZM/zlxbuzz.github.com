  • nginx 常用笔记

    • root 和 alias 的区别

    # nginx 常用笔记

    # root 和 alias 的区别

    # 当链接为localhost/a/,访问的是/www/a/下面的文件
location /a/ {
  root /www/a/;
}

    # 当链接为localhost/a/,访问的是/www/下面的文件
location /a/ {
  alias /www/a/;
}

    alias 会把 location 的路径给忽略，root 可以在 location 外部，而 alias 必须要 location 内部，建议都放到 location 中
