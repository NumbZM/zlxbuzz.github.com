  • nvm

    • 配置脚本
      • 执行

      # nvm

      由于开发机有系统默认的 node，n 安装的 node，nvm 安装的 node，导致很混乱，所以做一次整理，针对的是系统全局的安装。

      # 安装

      # 切换到root用户

mkdir /usr/local/nvm #定义安装目录
mkdir /opt/nvm #激活目录

git clone https://github.com/nvm-sh/nvm.git /opt/nvm

      # 配置脚本

      vim /etc/profile.d/nvm.sh

      export NVM_DIR=/usr/local/nvm # 导出NVM_DIR 环境变量，用来安装node
source /opt/nvm/nvm.sh # 执行nvm 的命令到每个用户的shell 中

nvm use v11.15.0 #指定版本

      # 执行

      nvm install v11.15.0

      重新登录，即可使用 node 和 npm
