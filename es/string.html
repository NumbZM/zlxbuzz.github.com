# String

# String.prototype.replace

# 对字符串进行替换,或者对匹配正则的字符串进行替换
str.replace(regexp/substr,replacement);
第一个参数：正则/字符串,第二个参数：字符串/函数
'abc'.replace('b', 2); //a2c
'abc'.replace(/bc/, 23); // a23

如果第二个参数有$开头，有特殊的作用:
$1、$2、...、$99 与 regexp 中的第 1 到第 99 个子表达式相匹配的文本。
'ababab'.replace(/a(b)/,'$12') // b2abab ,$1代表第一个子匹配b，也就是将ab替换成b2
$&	与 regexp 相匹配的子串。
'ababab'.replace(/a(b)/,'$&2') // ab2abab ,$&代表匹配的ab，也就是将ab替换成ab2
$` 位于匹配子串左侧的文本。
'ababab'.replace(/a(b)/,'$`2') // 2abab ,$`代表匹配的ab的左侧为空''，也就是将ab替换成2
$'	位于匹配子串右侧的文本。
'ababab'.replace(/a(b)/,'$\'2') // abab2abab ,$'代表匹配ab的右侧为abab，也就是将ab替换成abab2
$\$ 直接量符号。
'ababab'.replace(/a/,'$\$2') // $2babab, 直接替换成$2


如果第二个参数是函数:
一个匹配，（匹配项，字符串位置，原始字符串）
'abcabc'.replace(/a/,function(x,y,z){
    console.log(x,y,z);
    // a 0 abcabc
    return 1
})
// 1bcabc
多个匹配，（匹配项，...多个子匹配项,字符串位置，原始字符串）
'abcabc'.replace(/a(b)/g,function(x,y,z,xx){
    console.log(x,y,z,xx);
    //ab b 0 abcabc
    //ab b 3 abcabc
    return 1
})
// 1c1c
